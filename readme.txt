== Installation ==
1. Go to Plugins > Add New.
2. Search for \"webiots teamshowcase\".
1. Click \"Install Now\".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin. Look for \"Team Showcase\" in the admin menu.
